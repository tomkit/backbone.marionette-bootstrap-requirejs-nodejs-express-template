#Boilerplate
Boilerplate for starting a new node.js project with the following frameworks:

-Backbone.marionette
<br>
-Twitter Bootstrap
<br>
-RequireJS
<br>
-Express
<br>
##Up and Running

-git clone git@github.com:tomkit/backbone.marionette-bootstrap-requirejs-nodejs-express-template.git mynewrepo
<br>
-cd mynewrepo
<br>
-npm install
<br>
-node server.js
